CSocketMaster: RecvData

RecvData

This function is to retrieve data from the buffer. If we are using TCP then the data is retrieved from a local buffer (m_strRecvBuffer). If we are using UDP the data is retrieved from winsock buffer. This function returns number of bytes received, and move data from the buffer into the data argument. MOVE means that data will be removed from the buffer.
